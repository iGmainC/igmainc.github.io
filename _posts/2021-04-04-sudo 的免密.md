---
title: sudo 的免密
tags: Linux
---

在使用Linux系统过程中，通常情况下，我们都会使用普通用户进行日常操作，而root用户只有在权限分配及系统设置时才会使用，而root用户的密码也不可能公开。普通用户执行到系统程序时，需要临时提升权限，而有些提权操作会很频繁，直接`sudo`会要求输入密码，很麻烦，这时我们就可以通过修改`/etc/sudoers`来实现免密`sudo`

> `/etc/sudoers`是系统级配置，建议修改`/etc/sudoers.d/`下的配置文件，这是用户级配置
> 
> `sudoers`需要用`visudo`命令编辑，不能以 root 用户强行编辑

# Sudoers语法

```
用户  host=命令1,命令2
```

用户可以在`host`主机上执行`命令1`和`命令2`
**`命令1`和`命令2`必须写绝对目录，不然配置会失效**

```
%用户组  host=(root) 命令1,命令2
```

用户组在`host`主机上可以以`root`身份运行`命令1`和`命令2`
**`命令1`和`命令2`必须写绝对目录，不然配置会失效**

```
%用户组  host=(root) NOPASSWD: 命令1,命令2
```

用户组在`host`主机上可以以`root`身份且不需要密码运行`命令1`和`命令2`
**`命令1`和`命令2`必须写绝对目录，不然配置会失效**
